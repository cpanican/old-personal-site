The Adventures of Fluxion

Description

I developed this game using RPG Maker. The program provides a huge selection of character art, maps, backgrounds, music, dialog boxes, etc. All the user had to do is create a storyline, and modify the functions of every objects in the game. The user can still modify the raw code, or provide their own art (just like in the main menu screen).
