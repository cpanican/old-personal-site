Meds Search Engine

Description

During my summer internship of 2017, I created a Meds Search Engine for a startup company called pulseData. The search engine looks up a drug name and returns its ingredients and known side effect. The tool is not available to the public since it is an internal company tool.

I used different technologies to accomplish my task. I used Flask framework for the backend with Python. For the front-end I used HTML, CSS, and Flask-Bootstrap. To access the data, I used MySQL and then I then I implemented Apache Solr for free text searches. Finally, I placed everything on a Docker container and then the project was pushed to Amazon Web Services.
